Recomendador de recetas

Este programa ayuda a elegir recetas según las preferencias de sabor del usuario (dulce o salado) y proporciona los ingredientes y procedimientos para preparar los platos seleccionados.

1-Al ejecutar el programa, se dará la bienvenida y se le pedirá al usuario que ingrese su nombre. 2- Después de ingresar el nombre, se preguntará si se desea preparar una receta dulce o salada (debe ingresar 1 para dulce o 2 para salada). 3- Si se elige 1, las opciones son: alfajor de maicena o lemon pie. 4- Si se elige 2, las opciones son: tarta de calabaza o pastel de papa. 5- Después de seleccionar una receta, se verá una lista de ingredientes y el procedimiento para prepararla. 6- Existe una opción para agregar más ingredientes a la receta si se desea. 7- Una vez que se ha terminado con una receta, el programa lo llevará de vuelta al inicio, permitiendole elegir nuevamente un sabor y una receta.
